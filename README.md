# slowme-support
Slow Me - Apple App Store

Slow Me is a personal reflection app designed to help users understand themselves through gentle daily check-ins and fortnightly reflections.

## Support
If you need help or have feedback, please contact:
slowme.supportteam@icloud.com

The app is designed to work offline and does not require account creation or data submission.  
If you experience unexpected behaviour, please ensure you are using the latest version of the app.

Thank you for using Slow Me.
